Install in about a minute

No build, no sign-up. Load it and go.

Download the latest web-notes.zip from Releases and unzip it.
Open chrome://extensions and turn on Developer mode (top-right).
Click Load unpacked and pick the unzipped folder.
Pin the icon, open any site, and start noting. That's it.
